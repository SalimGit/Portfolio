RAPPORT DE STAGE

Dans le cadre de ma formation en BTS Services Informatiques aux Organisations. J’ai effectué un stage pour Iboski durant dix semaines, du 25 novembre 2019 au 3 février 2020 dont le but était la création d'un site de livraison de repas.

IBOSKI

IBOSKI est une société qui accompagne les porteurs de projets dans la création et le développement de leurs activités en proposant des solutions digitales. La société se complète pour former une équipe de 3 passionnés aux parcours très différents qui regroupent leurs expertises dans le but d'accompagner chaque idée jusqu'à sa réalisation et la satisfaction de la clientèle. IBOSKI aide les entreprises à soigner leurs identité visuelle et leurs présence sur le web dans le but d'atteindre divers objectifs: fidélisation, croissance du taux de conversion, trafic, publicité ciblée, image de marque référencement. OKmanger et Skyplus font partie de ses sites web.

Outil

Outils utilisé pour la réalistion du projet.

PHP

Principalement utilisé pour produire des pages Web dynamiques, PHP est un langage impératif orienté objet.

Laravel

C'est un framework web open-source écrit en PHP respectant le principe modèle-vue-contrôleur.

Bulma

Bulma est un framework CSS.

Visual Studio Code

Visual Studio Code est un éditeur de code extensible développé par Microsoft pour Windows, Linux et macOS (IDE).

Trello

Trello est un outil de gestion de projet, il repose sur une organisation des projets en planches représentant des tâches.

Discord

Discord est un logiciel conçu pour communiquer.

GitHub

GitHub est un service web d'hébergement et de gestion de développement de logiciels.

Javascript

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives.
